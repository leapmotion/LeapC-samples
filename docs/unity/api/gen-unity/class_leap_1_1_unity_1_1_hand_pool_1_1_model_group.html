Detailed Description

ModelGroup contains a left/right pair of IHandModel's.

Parameters
modelListThe IHandModels available for use by HandRepresentations
modelsCheckedOutThe IHandModels currently in use by active HandRepresentations
IsEnableddetermines whether the ModelGroup is active at app Start(), though ModelGroup's are controlled with the EnableGroup() & DisableGroup methods.
CanDuplicateAllows a IHandModels in the ModelGroup to be cloned at runtime if a suitable IHandModel isn't available.
