Detailed Description

A physics IHandModel implementation that works with the Interaction Engine to provide natural and subtle interaction between hands and physically simulated virtual objects.

Use the InteractionBrushHand models when using the Interaction Engine instead of the other physics hand models.

Since
4.1.3
