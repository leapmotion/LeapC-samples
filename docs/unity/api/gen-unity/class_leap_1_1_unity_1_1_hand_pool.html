Detailed Description

HandPool holds a pool of IHandModels and makes HandRepresentations when given a Leap Hand and a model type of graphics or physics.

When a HandRepresentation is created, an IHandModel is removed from the pool. When a HandRepresentation is finished, its IHandModel is returned to the pool.
