Enumeration Type Documentation

enum MessageSeverity

Reports whether the message is for a severe failure, a recoverable warning, or a status change.

Since
3.0
Enumerator
MESSAGE_UNKNOWN 

Unknown severity, indicates an error.

The rest of the data may be invalid.

MESSAGE_CRITICAL 

A problem severe enough to stop tracking.

MESSAGE_WARNING 

A correctable issue or status that can impact tracking.

MESSAGE_INFORMATION 

A verbose, informational message.
