Detailed Description

ISuspensionController defines the interface used by the Interaction Engine when the hand holding an interactable object ceases to be tracked and when it starts to be tracked again.

Tracking of a hand can be lost if it passes out of the Leap device's field of view or is otherwise blocked from view.

The Interaction Engine provides the SuspensionControllerDefault implementation for this controller.
