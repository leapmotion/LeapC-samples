Detailed Description

Allows you to add to a capped-size ring buffer of Ts and, when full, compute the buffer's average change over time.

DeltaBuffer without type parameters supports Vector3s; DeltaFloatBuffer supports floats. To support other types, subclass DeltaBuffer<YourType> and implement its Delta() averaging function.
